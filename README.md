# Post IPO Monitor

2년 이내 신규상장 종목의 수급/거래량/RSI/변동성 모니터링 시스템

## 실행 방법

```bash
cd post_ipo
python run.py
```

## 스코어링 시스템

### 전체 구조

```
종합스코어 = 모멘텀(30%) + 수급(50%) + 거래량(20%)
```

---

### 1. 모멘텀 스코어 (30%)

세 가지 요소의 **단순 평균**으로 계산:

| 요소 | 계산 방식 | 점수 범위 |
|------|----------|----------|
| 20일 수익률 | 백분위 순위 × 100 | 0~100 |
| 10일선 돌파 | 돌파=100, 미돌파=0 | 0 또는 100 |
| RSI 상태 | 구간별 고정 점수 | 20~80 |

#### RSI 점수 로직

```
RSI < 30  → 20점 (과매도 - 위험)
RSI > 70  → 30점 (과매수 - 위험)
RSI 30~49 → 60점 (중립)
RSI 50~70 → 80점 (강세 추세)
```

---

### 2. 수급 스코어 (50%)

기관과 외국인 **각각 가중평균** 후 평균:

```
기관 스코어 = 일간(40%) + 5일(35%) + 20일(25%)
외인 스코어 = 일간(40%) + 5일(35%) + 20일(25%)
수급 스코어 = (기관 + 외인) / 2
```

- 각 항목은 **백분위 순위** 기반 (순매수 금액이 높을수록 높은 점수)
- 최근 데이터(일간)에 더 높은 가중치 부여

---

### 3. 거래량 스코어 (20%)

```
RVOL(20)의 백분위 순위 × 100
```

- RVOL(20) = 당일 거래량 / 20일 평균 거래량
- RVOL = 1.0 → 평균 수준, RVOL = 2.0 → 평균의 2배
- 거래량이 평균 대비 높을수록 높은 점수

### 4. 변동성

```
VOLATILITY_30D (Bloomberg BDP)
```

- 30일 역사적 변동성 (Historical Volatility)
- 과거 30일 일간 수익률의 표준편차를 연율화한 값 (%)
- 값이 클수록 가격 변동이 심함

---

### 4. 등급 기준

| 등급 | 종합스코어 | 해석 |
|------|-----------|------|
| **A** | 80 ~ 100 | 매우 우수 |
| **B** | 65 ~ 80 | 우수 |
| **C** | 50 ~ 65 | 보통 |
| **D** | 30 ~ 50 | 주의 |
| **F** | 0 ~ 30 | 부진 |
| **-** | NaN | 데이터 부족 |

---

### 계산 예시

```
종목 X:
├── 모멘텀 스코어
│   ├── 20일수익률: 상위 30% → 70점
│   ├── 10일선 돌파: O → 100점
│   └── RSI: 55 → 80점
│   └── 모멘텀 = (70+100+80)/3 = 83.3
│
├── 수급 스코어
│   ├── 기관: (80×0.4 + 60×0.35 + 50×0.25) = 65.5
│   └── 외인: (70×0.4 + 65×0.35 + 60×0.25) = 65.75
│   └── 수급 = (65.5 + 65.75) / 2 = 65.6
│
└── 거래량 스코어
    └── RVOL(2.5) 상위 10% → 90점

종합 = 83.3×0.3 + 65.6×0.5 + 90×0.2
     = 25.0 + 32.8 + 18.0
     = 75.8 → B등급
```
