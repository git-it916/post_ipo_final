# Post IPO 모니터링 시스템

> 신규 상장 종목의 투자 매력도를 매일 자동으로 평가하고 순위화하는 내부 분석 도구

---

## 개요

국내 증시에 **최근 2년 이내 신규 상장한 종목**을 대상으로, 매일 장 마감 후 자동으로 시장 데이터를 수집하여 종합 점수를 산출하고 Excel 리포트를 생성합니다.

수급(기관·외국인 매매 동향), 모멘텀(가격 추세), 거래량(시장 관심도) 세 가지 관점에서 종목을 정량 평가하여 담당자의 투자 아이디어 발굴을 지원합니다.

---

│                                                             │

## 실행 방법

매일 **장 마감 후** 아래 명령어 한 줄로 실행합니다.

```
python run.py
```

실행 전 체크리스트:
- [ ] Bloomberg Terminal 실행 중
- [ ] `수급.xlsx` 당일 데이터로 업데이트
- [ ] `최초상장일.xlsx` 최신 버전 유지

## 분석 대상 종목

| 조건 | 내용 |
|------|------|
| 상장 시기 | 오늘 기준 **2년(730일) 이내** 신규 상장 종목 |
| 자동 제외 | ETF, ETN, 스팩(SPAC), 리츠, 기업인수목적회사 |
| 대상 범위 | 코스피·코스닥 보통주 (평균 약 150~170개 종목) |

> 종목 리스트는 `최초상장일.xlsx` 기준으로 매일 자동 산출되며, 별도 관리가 필요 없습니다.

## 종합 점수 산출 방식

세 가지 평가 영역을 가중 합산하여 **0~100점** 종합 점수를 부여합니다.

```
종합 점수 = 수급 (50%) + 모멘텀 (30%) + 거래량 (20%)
```

### 수급 점수 (50%) — 기관·외국인 매매 동향

기관과 외국인의 순매수 흐름을 단기·중기로 나누어 평가합니다.

| 기간 | 가중치 | 의미 |
|------|--------|------|
| 일간 | 40% | 당일 매매 집중도 |
| 5일 | 35% | 단기 추세 |
| 20일 | 25% | 중기 방향성 |

> 기관과 외국인이 동시에, 그리고 지속적으로 매수하는 종목일수록 높은 점수를 받습니다.

### 모멘텀 점수 (30%) — 가격 추세

주가의 방향성과 기술적 위치를 평가합니다.

| 항목 | 의미 |
|------|------|
| 20일 수익률 | 최근 한 달간 주가 상승 강도 |
| 10일선 돌파 여부 | 단기 이동평균 위에 있는지 |
| RSI | 현재 주가의 과열·침체 여부 (50~70 구간이 이상적) |

### 거래량 점수 (20%) — 시장 관심도

`RVOL(20)` 지표를 사용합니다. 당일 거래량이 최근 20일 평균 대비 몇 배인지를 나타냅니다.

| RVOL 값 | 의미 |
|---------|------|
| 1.0 | 평균 수준 |
| 2.0 | 평균의 2배 (관심 집중) |
| 5.0 이상 | 이슈 발생 가능성 |

## 등급 체계

| 등급 | 점수 범위 | 의미 |
|------|----------|------|
| **A** | 80점 이상 | 매우 우수 — 적극 검토 |
| **B** | 65 ~ 80점 | 우수 — 관심 유지 |
| **C** | 50 ~ 65점 | 보통 — 추가 모니터링 |
| **D** | 30 ~ 50점 | 주의 |
| **F** | 30점 미만 | 부진 |

## 결과 리포트 구성

저장 위치: `POST IPO 결과 / ipo_monitoring_YYYYMMDD.xlsx`

| 시트 | 내용 | 활용 |
|------|------|------|
| **추천종목** | 종합 점수 상위 50개 | 매일 첫 번째로 확인 |
| **A등급** | 80점 이상 종목 전체 | 집중 검토 대상 |
| **전체** | 전체 분석 종목 데이터 | 개별 종목 조회 |
| **RSI과매수** | RSI 70 초과 종목 | 단기 차익 구간 주의 |
| **RSI과매도** | RSI 30 미만 종목 | 저점 매수 검토 |
| **외국인순매수TOP** | 외국인 당일 매수 상위 20 | 외국인 수급 동향 |
| **기관순매수TOP** | 기관 당일 매수 상위 20 | 기관 수급 동향 |
| **RVOL_TOP** | 거래량 급증 상위 20 | 이슈 종목 조기 탐지 |
| **RSI추이** | RSI 과열 종목수 일별 변화 | 시장 전체 과열도 판단 |

## 추천종목 시트 주요 컬럼 설명

| 컬럼 | 설명 |
|------|------|
| 종합스코어 | 0~100점, 높을수록 우수 |
| 모멘텀스코어 | 가격 추세 점수 |
| 수급스코어 | 기관·외국인 매매 점수 |
| 거래량스코어 | 거래량 집중도 점수 |
| RSI(14) | 현재 과열·침체 지수 (50~70이 이상적) |
| RSI변화량 | 전일 대비 RSI 변화 (+ 면 추세 강화) |
| RVOL(20) | 당일 거래량 / 20일 평균 거래량 |
| 기관_일간 | 기관 당일 순매수 (단위: 백만원) |
| 외국인_일간 | 외국인 당일 순매수 (단위: 백만원) |
| 변동성(30D) | 최근 30일 연율화 변동성 (낮을수록 안정) |

---

## RSI추이 시트 — 시장 과열도 모니터링

전체 분석 종목 중 RSI 65 이상(단기 과열 구간) 종목의 수를 매일 기록하여 시장 전반의 온도를 추적합니다.

| 컬럼 | 설명 |
|------|------|
| 날짜 | 기준일 |
| 분석종목수 | 당일 데이터 수집 가능 종목 수 |
| RSI65이상 | RSI 65 초과 종목 수 |
| 과매수비율(%) | 전체 대비 비율 |
| 전일대비 | 전일 대비 증감 |

> 과매수비율이 지속 상승하면 신규 IPO 종목 전반의 단기 과열 신호로 해석할 수 있습니다.

## 데이터 기준 시점

| 데이터 종류 | 기준 |
|------------|------|
| 종가·거래량 | 전 영업일 (T-1) |
| 기관·외국인 수급 | 수급.xlsx 최신 업데이트일 |
| RSI·변동성·이평선 | Bloomberg 실시간 제공값 |

> 수급.xlsx가 당일 기준으로 업데이트되지 않은 경우, 실행 시 경고 메시지가 표시됩니다.
